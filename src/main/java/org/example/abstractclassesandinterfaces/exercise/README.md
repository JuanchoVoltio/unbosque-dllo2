Deberán desarrollar una calculadora que permita las operaciones aritméticas básicas.

Equipo A: La calculadora le permitirá al ausuario hacer el cálculo de áreas de figuras geométricas regulares (triángulo, círculo, rectángulo).
Equipo B: La calculadora le proveerá al usuario de funcionalidades para calcular el volúmen de una figura regular (Cono, Cubo, Esfera).
Equipo C: La calculadorá deberá permitir el uso de una memoria temporal que deberá estar disponible para futuras operaciones.
